# 项目：个人智能体管理平台
# 功能
## 用户管理
- 用户注册
- 用户登录
- token认证
## 智能体管理
- 智能体创建
- 智能体列表
- 智能体修改（提示词、描述）
- 智能体删除
## 会话管理
- 会话创建
- 会话列表
- 会话详情
- 会话删除
## 消息管理
- 消息发送
- 消息查询
- 消息列表

# 用户体验
用户可以在这个平台上注册账号，登录后创建、修改、删除智能体，点击智能体头像或名称可与智能体进行对话。

# 部署
## 环境要求
- Docker
- Docker Compose
## 部署步骤
1. 克隆项目代码：`git clone https://github.com/Xcqqqqq/devops2025.git`
2. 进入项目目录：`cd devops2025`
3. 进入部署目录：`cd deploy`
4. 复制环境变量配置文件：`cp .env.example .env`
5. 修改`.env`文件中的配置，包括数据库密码、AI_API_KEY等
6. 一键部署：`./deploy.sh` 或者 `make deploy`
